Control Flow Nodes

These nodes manage the logic and path of your workflow.

If/Else Node

Routes execution based on a Python condition. Example: int(score) > 8.

While Loop

Repeats a branch as long as a condition is true. Essential for self-correction loops.

JSON Dispatcher

Extracts a key from JSON (e.g., "action") and routes to handles named after the possible values (e.g., "search", "write").

Sequential Batch

Iterates through a list of items sequentially. Perfect for executing multi-step plans generated by an LLM.

Sub-Workflow

Executes another saved workflow as a single step. Allows for modular and hierarchical design.
