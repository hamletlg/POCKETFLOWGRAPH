Advanced Developer Tools

These nodes provide powerful capabilities for users who want to extend the platform with custom code.

1. Script Execution Node

Run arbitrary Bash commands or Python scripts directly in your workflow.

⚠️ Security Warning: This node allows arbitrary code execution on your system. Only run scripts that you trust.

Parameters:

interpreter bash or python.
script_body The actual code to run. (Example: print(2 + 2))
input_mode How to receive data from the previous node: stdin (Standard Input), env (via $POCKETFLOW_INPUT variable), or none.
return_type What the node outputs: stdout (text result), exit_code (0 for success), or json (a full object with stdout/stderr/code).
Example (Python):
Interpreter: "python"
Script Body: print("Result: " + input())
Input Mode: "stdin"
Expected Result: If the previous node sent "Hello", this node returns "Result: Hello".
