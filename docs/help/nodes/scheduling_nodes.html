Web & Automation

These nodes connect your agent to the internet and let it work even while you sleep.

1. Web Search

Searches the live internet for news, facts, or answers.

Parameters:

query What you want to find. (Example: Weather in London)
max_results How many links to find. (Example: 5)
as_list True/False. If true, outputs a list of items for a "Loop" node.
Example:
Query: "NVIDIA stock price yesterday"
Expected Result: A list of links and short summaries about the stock price.

2. Page Reader (Web Fetch)

Clicks a link and reads all the text on that specific webpage.

Parameters:

url The address of the website. (Example: https://example.com)
max_chars Limit the amount of text (to save AI costs). (Example: 5000)
Example:
URL: "https://wikipedia.org/wiki/AI"
Expected Result: The full text content of that Wikipedia article.

3. Schedule (Cron)

Tells the workflow to start itself automatically at certain times.

Parameters:

schedule_type "Interval" (every X mins) or "Cron" (exact time).
cron_expression Standard time format. (Example: 0 9 * * * for 9 AM daily)
interval_value The number for the interval. (Example: 30)
interval_unit "Seconds", "Minutes", or "Hours".
Example:
Type: "Interval"
Value: "1"
Unit: "Hours"
Expected Result: Your workflow runs by itself every hour on the dot.
