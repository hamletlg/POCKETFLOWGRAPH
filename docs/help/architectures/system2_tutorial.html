System 2: "Deep Thinking"

Human brains have two ways of thinking: System 1 (fast, intuitive, sometimes wrong) and System 2 (slow, logical, careful). We can build agents that work the same way!

System 1
"What is 2+2?"
Instant answer: 4.
System 2
"What is 1,234 x 567?"
I need to write this down and double-check my math.

How to build a "Careful" Agent:

  1. The Generator: An LLM node writes a first draft of an answer.
  2. The Critic: A second LLM node is told: "Find every mistake in that draft."
  3. The Decision: Use a While Loop. If the Critic found mistakes (Score < 8), send the draft back to the Generator to fix it.
  4. Finalization: Only output the result once it passes the Critic's test.

Why bother?

This "Thinking Out Loud" or "Self-Correction" process significantly reduces AI mistakes (hallucinations) and results in much higher quality writing or coding.
