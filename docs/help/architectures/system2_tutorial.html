System 2 Reasoning / Reflexion

System 2 thinking involves slowing down the agent to critique and refine its own output before finalizing it.

The Architecture

  1. Generator: An LLM node that produces the initial draft.
  2. Critic: A second LLM node prompted to look for flaws in the draft.
  3. Variable Extractor: Parses the Critic's feedback (e.g., getting a "quality_score").
  4. While Loop: Checks if quality_score < 8. If so, it loops back to the Generator.

Benefits

  • Higher quality output for complex tasks.
  • Reduced hallucinations through self-verification.
  • Mimics human-like "deliberate" thinking.

Example Workflow: Check reasoning_loop.json in the examples folder.
