Autonomous Planning

Planners decompose a large, vague goal into a structured list of actionable steps.

The Architecture

  1. Planner LLM: Outputs a list of sub-tasks in JSON.
  2. Variable Extractor: Pulls the task_list from the LLM output.
  3. Sequential Batch: Iterates through the task_list.
  4. Executor: Processes each task (e.g., using Sub-Workflow).

Key Benefits

  • Handles extremely large tasks that exceed the LLM's single-shot capability.
  • Allows for "Checkpoints" where progress can be saved.

Example Workflow: Check autonomous_planner.json in the examples folder.
