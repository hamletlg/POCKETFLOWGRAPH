ReAct: Thinking + Acting

A ReAct agent is like a researcher who thinks about what they need to know, uses a tool to find it, and then thinks again based on what they found.

How the Loop Works:

1. Reason (Think)
The LLM "Brain" looks at the goal and thinks: "I need to know the price of gold to answer this."
2. Act (Tool Use)
The Brain decides to use the Web Search node.
3. Observe (Learn)
The Web Search node returns the price. This info is saved in Memory.
4. Repeat or Finish
The Brain looks at the new info. If it can answer the question, it finishes. If not, it loops back to step 1.

Key Nodes You'll Need:

  • LLM Node: To do the "Reasoning."
  • JSON Dispatcher: To help the Brain click the right "Action" button.
  • Web Search: To act as the agent's eyes on the internet.
  • Memory Node: To keep track of what has been discovered so far.
