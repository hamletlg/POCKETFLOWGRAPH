ReAct: Reasoning + Acting

A ReAct agent combines logical reasoning with tool use (actions) to solve complex problems.

The Loop

  1. Reason: LLM thinks about the next step.
  2. Act: LLM chooses a tool (e.g., Web Search) via JSONDispatcher.
  3. Observe: The tool's output is fed back into memory.
  4. Repeat: The agent continues until the goal is met.

Key Nodes

  • JSON Dispatcher: Routes the "Action" to the correct node.
  • Web Search: Gathers external facts.
  • Memory Node: Keeps track of the search results for the next reasoning step.

Example Workflow: Check react_agent.json in the examples folder.
